# Testing workshop

## Kom i gang

### Klon repoet

```
git clone git@github.com:erlenlo/javascript-testing-workshop.git
```

### Installering av pakker

Naviger inn i repoet og kjør kommandoen

```
npm install
```

### Start prosjektet

```
npm start
```

### Kjøring av tester

Jest-tester kjøres med

```
npm run test
```

Cypress tester kjøres med

```
npm run cypress
```
